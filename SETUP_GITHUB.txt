Melacs — eget repo (inte fork)
================================

Det här projektet är kopplat till:
  https://github.com/stefanmaston/SEAS-Melacs-Github-20260408

I git finns bara remote "origin" (inga upstream-remotes). Det är rätt för ett eget repo.

Om du skapar repot på GitHub igen:
  • Använd "New repository" — skapa INTE som fork av någon annan.
  • Tomt repo är OK; första push laddar upp allt.

Pusha från den här mappen (Terminal):

  cd "/Users/stefanlarsson-mastonstrale/Documents/Sunthetics/Websida/SEAS Melacs Github 20260408"
  git branch -M main
  git push -u origin main

Om GitHub frågar om inloggning:
  • GitHub Desktop, eller
  • Terminal:  gh auth login
  • Eller byt till SSH och lägg till nyckel i GitHub:
      git remote set-url origin git@github.com:stefanmaston/SEAS-Melacs-Github-20260408.git
      git push -u origin main

Efter lyckad push ska GitHub visa koden utan "forked from …".
