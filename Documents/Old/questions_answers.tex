\section{Questions and Answers}
\label{ch:questions_answers}

\vspace*{0.3in}

\begin{description}[style=nextline]
	\item[What to do when battery is completely discharged?] Use AC-grid to charge the battery when connected to AC grid. In off-grid mode use an external AC inverter fed from an DC source to charge the battery.
	\item[In container system, how do we radiate heat, since there is no radiator?] There can be cases where the battery SOC is low and hot water is sufficient. Since there is no radiator. It is the responsibility of container installer to make sure the extra heat is radiated.
	\item[How does the system know exact number of externally connected batteries?] The should be configurable item in the GUI. So user has to set it every-time they change the number of batteries.
	\item[Who is responsible for the hot water during the night time in solar tracker?] The user is responsible for the hot water during the night.
	
\end{description}
