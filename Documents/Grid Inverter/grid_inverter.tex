\section{Grid Inverter}
\label{ch:grid_inverter}

The 3-phase grid inverter supplies power directly to the grid (single or 3-phase) or to a local load which is off-grid. The maximum power capability of the system is 10kW. \newline

The grid inverter for this power generation system connects directly with the grid or local load and converts an input voltage DC source to single or 3-phase AC output. \newline
